Questionnaire Resource

The join of record in asmt_vrsn and asmt_sbst_rfrnc will become one Questionnaire resource in FHIR. The sections and questions will be represented as items within the questionnaire. Only active questions are migrated from the DEL.

Example: Hospice Item Set has two subsets (one for admission and one for discharge). This will result in two questionnaire resources: `HIS-HA-1.00.0` and `HIS-HD-1.00.0`

asmt_sbst_rfrnc.asmt_sbst_shrt_name		 Fields are joined using a dash (-) in the format {asmt_shrt_name}-{asmt_sbst_rfrnc.asmt_sbst_shrt_name}-{asmt_vrsn_id}. Example: LCDS-1.01.

effectivePeriod.end asmt_vrsn.efctv_end_dt Only populated if field value is not null.

item[] Populated with list of questions or list of sections.
